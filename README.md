# m-global-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容不针对任何特定域名或网站，而是作为通用资料存档与展示用途。

## 项目简介

`m-global-notes` 是一个轻量级的 HTML 页面集合仓库。所有页面均为独立文件，无需依赖后端服务或数据库。仓库主要目的是保存与组织各类笔记、参考信息或静态内容，方便后续查阅与分享。

## 目录结构

```
m-global-notes/
├── index.html          # 仓库首页或导航页
├── pages/              # 存放独立 HTML 页面的文件夹
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：可选的仓库入口页面，用于列出或链接到各归档页面。
- `pages/`：核心目录，所有独立 HTML 页面均放置于此。每个页面可包含不同的主题或内容。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文件，不依赖外部资源（除非在页面内显式引用）。
- 页面内容以信息归档、笔记整理或静态展示为主，不包含动态交互或用户数据收集。
- 归档内容可能随时间更新或补充，但不会删除已有页面，除非有重大调整。

## 维护说明

- 本仓库由维护者不定期更新，添加新的 HTML 页面或修正已有页面内容。
- 欢迎通过 Issues 提交建议或报告问题，但不保证即时响应。
- 不接受直接向仓库推送内容的外部贡献，除非经过明确沟通。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件，即可查看对应内容。
3. 也可通过 GitHub Pages 或任意静态服务器部署 `index.html` 作为入口。
